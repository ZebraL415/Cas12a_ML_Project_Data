# current

本目录保存当前仍在使用的活动记录。

## 文件

- `problems_to_resolve_zh.md` / `problems_to_resolve_en.md`：当前仍需确认的问题和已经解决的问题摘要。
- `meeting_decisions_zh.md` / `meeting_decisions_en.md`：会议或人工决策记录。
- `paper_data_notes_zh.md` / `paper_data_notes_en.md`：跨论文或跨数据源的数据备注。

## 使用原则

- 当前问题写在这里；历史运行证据写在 `../runs/`。
- 解决一个问题后，应保留原问题、补充解决结论，并说明证据来源。
- 不要在这里存放训练数据或原始表格。
