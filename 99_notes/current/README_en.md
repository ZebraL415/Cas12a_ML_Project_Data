# current

This directory stores active records that are still in use.

## Files

- `problems_to_resolve_zh.md` / `problems_to_resolve_en.md`: current unresolved questions and summaries of resolved issues.
- `meeting_decisions_zh.md` / `meeting_decisions_en.md`: meeting notes or human decisions.
- `paper_data_notes_zh.md` / `paper_data_notes_en.md`: cross-paper or cross-source data notes.

## Usage rules

- Current questions go here. Historical evidence from a specific run goes in `../runs/`.
- When a question is resolved, keep the original question, add the conclusion, and cite the evidence source.
- Do not store training data or raw tables here.
